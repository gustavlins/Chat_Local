## Local Network Chat

This project implements a simple local chat application built with **JavaScript** and **Node.js**, allowing communication between devices within the same network using an IP address.

The application creates a lightweight server that listens for incoming connections and enables message exchange between connected clients in real time.

### Features

- Real-time messaging between users
- Local network communication using IP address
- Lightweight server built with Node.js
- Simple terminal-based interface
- Easy setup and execution

### Technologies

- JavaScript
- Node.js
- TCP/Socket communication

### Purpose

The main goal of this project is to explore **network communication concepts**, **server-client architecture**, and **real-time messaging** using Node.js in a local environment.

### How It Works

1. The server is started using Node.js and listens on a specific IP and port.
2. Clients connect to the server using the network IP address.
3. Messages sent by one client are transmitted to the server and broadcast to other connected clients.

This project serves as a practical example for understanding **network programming**, **socket communication**, and **basic distributed systems concepts**.
